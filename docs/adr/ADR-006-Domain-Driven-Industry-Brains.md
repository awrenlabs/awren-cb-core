# ADR-006: Domain-Driven Industry Brains

**Status**: Approved
**Date**: 2026-06-06

## Context
Awren Core needs to operate across multiple industry verticals.

## Decision
Pluggable domain architecture where each domain defines its own ontology, entities, agents, and rules.

## Consequences
- Clean separation of domain concerns
- Cross-domain queries are complex
- Reusable core infrastructure
