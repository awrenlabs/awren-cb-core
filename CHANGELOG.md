# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-06

### Added

- **Core Entity Model**: BaseEntity, BaseRelationship, BaseEvent classes with Pydantic validation
- **Ontology Engine**: OWL 2 ontology management, SHACL validation framework
- **Memory Engine**: Event-sourced memory architecture with episodic, semantic, and procedural memory types
- **Reasoning Engine**: Multi-modal reasoning pipeline (deductive, inductive, abductive, analogical)
- **Agent Framework**: Multi-agent orchestration with specialized agent types
- **Ingestion Pipeline**: Document and data ingestion framework
- **Observability**: OpenTelemetry integration, Prometheus metrics, structured logging
- **Python SDK**: External developer SDK for programmatic access
- **FastAPI Application**: REST API with auto-generated OpenAPI documentation
- **CLI Tool**: Command-line interface for development and operations
- **Construction Domain**: First vertical implementation ontology and logic
- **Docker Infrastructure**: Containerized PostgreSQL, Neo4j, Qdrant
- **GitHub CI/CD**: Automated linting, testing, building, and documentation workflows
- **Documentation**: Full ADRs (10), RFCs (10), engineering standards, research papers
