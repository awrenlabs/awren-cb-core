# Awren Core Roadmap

## R1: Foundation (Q3-Q4 2026)

**Goal**: Establish the core data model and ontology infrastructure.

- [ ] Core entity model (BaseEntity, BaseRelationship, BaseEvent)
- [ ] Ontology registry with OWL 2 support
- [ ] Basic knowledge graph storage (Neo4j)
- [ ] SPARQL query interface
- [ ] Document ingestion pipeline
- [ ] FastAPI application shell

## R2: Ontology (Q4 2026)

**Goal**: Full ontology management with validation and reasoning.

- [ ] SHACL validation framework
- [ ] Ontology versioning and diff
- [ ] RDFS/OWL RL reasoning
- [ ] Ontology alignment and mapping
- [ ] Web-based ontology explorer

## R3: Memory (Q1-Q2 2027)

**Goal**: Persistent, event-sourced memory architecture.

- [ ] Append-only event log
- [ ] Episodic memory (event-based)
- [ ] Semantic memory (factual knowledge)
- [ ] Procedural memory (processes and workflows)
- [ ] Temporal query interface
- [ ] Memory consolidation and pruning

## R4: Reasoning (Q2-Q3 2027)

**Goal**: Multi-modal reasoning over knowledge graph.

- [ ] Deductive reasoning (OWL inference)
- [ ] Inductive reasoning (pattern discovery)
- [ ] Abductive reasoning (explanation generation)
- [ ] Analogical reasoning (cross-domain transfer)
- [ ] Natural language question answering
- [ ] Explainable reasoning traces

## R5: Agents (Q3-Q4 2027)

**Goal**: Multi-agent framework for distributed intelligence.

- [ ] Agent orchestrator with task decomposition
- [ ] Research agent for literature review
- [ ] Ontology agent for ontology management
- [ ] Domain agent for domain-specific Q&A
- [ ] Memory agent for retrieval and consolidation
- [ ] Decision agent for decision support

## R6: Construction Brain (Q4 2027)

**Goal**: First vertical implementation.

- [ ] Construction ontology (projects, contracts, resources)
- [ ] Construction-specific entity types
- [ ] Risk analysis and prediction agents
- [ ] Document understanding for construction docs
- [ ] Integration with construction data sources

## R7: Company Brain (2028)

**Goal**: Horizontal platform for any organization.

- [ ] Multi-tenant architecture
- [ ] Organization onboarding and configuration
- [ ] Cross-domain knowledge integration
- [ ] Enterprise SSO and access control
- [ ] Admin dashboard and analytics

## R8: Multi-Industry Platform (2029+)

**Goal**: Platform for multiple industry verticals.

- [ ] Manufacturing domain
- [ ] Logistics and supply chain domain
- [ ] Government and public sector domain
- [ ] Healthcare domain
- [ ] Industry marketplace and plugin system
