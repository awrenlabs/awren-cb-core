# Awren Core — Observability Package

Observability infrastructure for the Awren Cognitive OS.

## Components
- **MetricsCollector**: Prometheus-compatible metrics
- **StructuredLogger**: JSON-structured logging
- **Tracer**: Distributed tracing (OpenTelemetry-compatible)

## Pillars
- Metrics: Quantitative measurements (Prometheus + Grafana)
- Logs: Structured event records
- Traces: Distributed request tracing
- Knowledge Quality: Ontology and data quality metrics
- Reasoning Audits: Explainable reasoning traces
