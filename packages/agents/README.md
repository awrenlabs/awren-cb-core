# Awren Core — Agents Package

Multi-agent orchestration framework for the Awren Cognitive OS.

## Agent Types
- **Research Agent**: Literature review, hypothesis generation
- **Ontology Agent**: Ontology management, mapping, validation
- **Domain Agent**: Domain-specific Q&A and analysis
- **Memory Agent**: Memory retrieval and consolidation
- **Decision Agent**: Decision support and analysis
- **Monitoring Agent**: System observability and alerts

## Architecture
- **AgentOrchestrator**: Central coordinator for task decomposition
- **BaseAgent**: Abstract base class for all agents
- **Message Bus**: Inter-agent communication protocol
